Quick start

Using CDN

You can easily start by using CDN.

Add the following description in the <head> tag of the web page you want to use Pax.

You can also bundle the library with Browserify/Webpack

npm install @appleple/pax-js --save
import Pax from '@appleple/pax-js'

Just replace the style="background-image:url(...)" to data-img="..."

<div class="js-parallax" 
  data-img="https://source.unsplash.com/pb_lF8VWaPU/1600x900">
  <p>contents here</p>
</div>

Write the following script

new Pax('.js-scrollable');
